How to Play Sequences

1. Find the Pattern

Look at the numbers shown and figure out the pattern (counting, skip counting, or something trickier!).

2. Fill in the Blanks

Use the number pad to type the missing number, then tap the check mark to submit.

3. Keep Going!

Each correct answer earns points. The patterns get harder as you go. One wrong answer ends the game!

4. Scoring

  • Easy patterns: 1 point each
  • Medium patterns: 2-3 points each
  • Hard patterns: 5-10 points each
